Learn CSS

Lessons

1.CSS Introduction

CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media

2.CSS Syntax

A CSS rule consists of a selector and a declaration block.

3.CSS Comments

CSS comments are not displayed in the browser, but they can help document your source code.

4.CSS Colors

Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

5.CSS Backgrounds

The CSS background properties are used to add background effects for elements.

6.CSS Borders

The CSS border properties allow you to specify the style, width, and color of an element's border.

7.CSS Margins


Margins are used to create space around elements, outside of any defined border

8.CSS Padding

Padding is used to create space around an element's content, inside of any defined borders.

9.CSS Height, Width

The CSS height and width properties are used to set the height and width of an element. The CSS max-width property is used to set the maximum width of an element.

10.CSS Text Alignment


The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified..

11.CSS Layout - The display Property

HTML headings are titles or subtitles that you want to display on a webpage.

12.CSS Fonts

Using a font that is easy to read is important. The font adds value to your text. It is also important to choose the correct color and text size for the font..

13.CSS Links


With CSS, links can be styled in many different ways.

14.CSS Lists

Set different list item markers for ordered lists. Set different list item markers for unordered lists

15.CSS Tables

The look of an HTML table can be greatly improved with CSS
